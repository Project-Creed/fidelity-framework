# Fidelity Book Index

This index organizes the manuscript source in `chapters/` into a recommended reading sequence.

## Recommended Reading Order

1. [Introduction](../chapters/00-introduction.md)
2. Read Part I through Part IV in numeric order.
3. Revisit chapters based on the relational domain you are facing now: love, trust, promise, loyalty, friendship, dating, sexuality, marriage, family, caregiving, conflict, forgiveness, boundaries, betrayal, estrangement, technology, or grief.
